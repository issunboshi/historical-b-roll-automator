2. **Add CLI arguments** using ArgumentDefaultsHelpFormatter:
   - `--min-priority` (float, default 0.5): Minimum priority threshold. 0.0 disables filtering.
   - `-v/--verbose` (store_true): Show per-entity skip messages.

3. **Create `should_skip_entity()` function** with guard clause pattern:
   ```python
   def should_skip_entity(
       entity_name: str,
       entity_data: Dict,
       min_priority: float,
       transcript_duration: float
   ) -> Tuple[bool, str]:
   ```

   Guard clauses in order:
   - If min_priority <= 0.0: return (False, "") -- filtering disabled
   - If entity_type == "people": return (False, "") -- always download
   - If entity_type == "events": return (False, "") -- always download
   - If entity_type == "places":
     - Calculate first_position_pct using srt_time_to_seconds
     - If first_pct <= 0.1: return (False, "") -- early mention override
     - If mention_count >= 2: return (False, "") -- multiple mentions override
     - If priority < min_priority: return (True, "place with N mention(s), not in first 10%")
   - If entity_type == "concepts":
     - If priority < 0.7: return (True, "concept priority X.XX < 0.70")
   - Default: If priority < min_priority: return (True, "priority X.XX < Y.YY")
   - Otherwise: return (False, "")

4. **Normalize entity_type to lowercase** in comparisons to handle casing inconsistencies.

5. **Extract transcript_duration** from enriched_entities.json metadata (or compute from last occurrence if not present).

   - After loading entities_map, iterate through entities
   - For each entity, call should_skip_entity()
   - Build two lists: `to_download` (passed filter) and `skipped_entities` (failed filter)
   - Use logger.info() for per-entity skip messages (only shown with -v)

3. **Skipped entities list** structure:
   ```python
   skipped_entities.append({
       "name": entity_name,
       "entity_type": entity_data.get("entity_type"),

       "mention_count": len(entity_data.get("occurrences", [])),
       "reason": skip_reason
   })
   ```

4. **Add skipped array to output JSON**:
   ```python
   entities_map["skipped"] = skipped_entities
   ```
